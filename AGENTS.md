# chai

Charlie's AI configuration repo — shared config for Claude and Gemini.

## Structure

- `_AGENTS.md` — shared agent definitions (installed globally via `install.sh`)
- `skills/` — shared skills
- `mcp/` — shared MCP server configs
- `claude/` — Claude-specific config
- `gemini/` — Gemini-specific config
- @CHAI_AGENTS - represents a symlink to GEMINI.md and CLAUDE.md at user level
